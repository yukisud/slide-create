# インストール手順（macOS）

Apple Developer 署名なしの場合は、初回のみ以下の操作が必要です。

1. DMGを開く
2. アプリを「アプリケーション」へドラッグ
3. Finderの「アプリケーション」からアプリを右クリック →「開く」
4. 警告が出たら「開く」を選択

もし「壊れているため開けません」と表示される場合:
1. `unblock-mac.command` をダブルクリック
2. Terminalが開いたら完了まで待つ
3. 以降は通常のダブルクリックで起動できます

補足:
- `unblock-mac.command` は「アプリケーション」に入れたアプリに対して実行してください
- 管理者権限が必要な場合はパスワード入力が求められます

一度開けば、以降は通常のダブルクリックで起動できます。
