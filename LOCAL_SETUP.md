# ローカル起動ガイド（1クリック）

GitHub Pagesの画面からファイルをダウンロードし、ローカルで起動できます。

## 手順

1. リポジトリをZIPでダウンロードして解凍します。
2. OSに合わせて以下をダブルクリックします。
   - Mac: `start-mac.command`
   - Windows: `start-win.bat`
3. ブラウザで `http://localhost:8010/index.html` が開きます。

## よくある質問

### Q. 開かない / 起動しない
- Pythonが未インストールの可能性があります。https://www.python.org/ からインストールしてください。
- 既に別のアプリが `8010` を使用している場合は、スクリプト内の `PORT=8010` を別の番号に変更してください。

### Q. 終了方法は？
- 起動したターミナル/コマンド画面を閉じれば停止します。
